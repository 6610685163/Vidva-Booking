# Vidva Booking

ระบบจองห้องประชุมและห้องเรียน ภาควิชาวิศวกรรมไฟฟ้าและคอมพิวเตอร์ ซึ่งพัฒนาด้วย Django Framework เพื่ออำนวยความสะดวกแก่อาจารย์และเจ้าหน้าที่
โดยถูกออกแบบมาเพื่อจัดการห้องพักจำนวน 5 ห้อง โดยรองรับการจอง การตรวจสอบตารางว่างผ่านปฏิทิน และระบบอนุมัติการจองที่เชื่อมต่อกับระบบยืนยันตัวตนของมหาวิทยาลัยธรรมศาสตร์

## ฟีเจอร์
* **TU REST API Login:** เข้าสู่ระบบด้วย TU Account ได้ทันที ไม่มีการเก็บรหัสผ่านไว้ในระบบเพื่อความปลอดภัย
* **Conflict Detection:** ตรวจสอบและแจ้งเตือนอัตโนมัติหากมีการจองห้องซ้ำซ้อนในช่วงเวลาเดียวกัน
* **Calendar View:** แสดงตารางการใช้ห้องแบบรายสัปดาห์และรายเดือน พร้อมแถบสีแยกสถานะ (ว่าง, รออนุมัติ, อนุมัติแล้ว)
* **Recurring Booking:** รองรับการจองแบบต่อเนื่อง (เช่น จองทุกวันจันทร์และพุธ) สำหรับการเรียนการสอน
* **Notification System:** ระบบส่งอีเมลแจ้งเตือนเจ้าหน้าที่เมื่อมีการจองใหม่ และแจ้งผู้จองเมื่อได้รับการอนุมัติ

## เทคโนโลยีที่ใช้
* **Backend:** Python / Django 5.0.4
* **Database:** SQLite (สำหรับ Development) / PostgreSQL
* **Frontend:** Responsive Design รองรับการใช้งานผ่านมือถือและคอมพิวเตอร์ (CSS Grid)
* **Authentication:** TU REST API

## ขั้นตอนการติดตั้งและเริ่มใช้งาน

1. **Clone และสร้าง Virtual Environment:**
   ```bash
   git clone <repository-url>
   cd TSE-Room-Booking
   python -m venv venv
   source venv/bin/activate  # สำหรับ Windows ใช้ venv\Scripts\activate
   ```

2. **ติดตั้ง Library ที่จำเป็น:**
   ```bash
   pip install -r requirements.txt
   ```

3. **ตั้งค่า Environment Variables:**
   คัดลอกไฟล์ `.env.example` ไปเป็น `.env` แล้วระบุค่าดังนี้:
   - `TU_API_KEY`: Application-Key ที่ได้จาก [https://restapi.tu.ac.th](https://restapi.tu.ac.th)
   - `EMAIL_HOST_USER` / `EMAIL_HOST_PASSWORD`: สำหรับระบบแจ้งเตือนผ่านอีเมล

4. **เตรียมฐานข้อมูล:**
   ```bash
   python manage.py migrate
   ```

5. **รันเซิร์ฟเวอร์:**
   ```bash
   python manage.py runserver
   ```

---
**หมายเหตุ:** สำหรับผู้ใช้ใหม่ที่ Login ครั้งแรก เจ้าหน้าที่ (Admin) จะเป็นผู้กำหนดสิทธิ์การใช้งาน (Role) ให้ในระบบจัดการผู้ใช้
